    <i class="fa fa-plus"></i>

    <button class="woodash-btn woodash-btn-secondary">Edit</button>

</div>

CSS Classes Reference

Button Variants

  • woodash-btn-primary - Main action button
  • woodash-btn-secondary - Secondary action
  • woodash-btn-success - Success/confirm action
  • woodash-btn-danger - Delete/destructive action
  • woodash-btn-warning - Warning/caution action
  • woodash-btn-info - Information action
  • woodash-btn-ghost - Subtle action
  • woodash-btn-outline - Outlined style

Button Sizes

  • woodash-btn-sm - Small size (32px height)
  • woodash-btn - Default size (44px height)
  • woodash-btn-lg - Large size (52px height)
  • woodash-btn-xl - Extra large size (60px height)
  • woodash-btn-icon - Icon only (44px × 44px)
  • woodash-btn-fab - Floating action button
